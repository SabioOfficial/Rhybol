
# Rhybol

A _*WIP*_ terminal-based rhythm game. Inspired by [Project: Rusher](https://www.roblox.com/games/106809703371658/Project-RUSHER) _(cool game!)_

For [TerminalCraft](https://terminalcraft.hackclub.com/) YSWS.

## How to run locally?

cd into the root directory then run ```python main.py``` (requires python)
